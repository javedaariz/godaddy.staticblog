# World's Best Tourist Spots Explorer

A dynamic web application that provides comprehensive travel information about the world's best tourist destinations, including visa requirements, accommodation options, safety information, and top attractions.

## Features

- Dynamic Destination Discovery Browse through curated tourist spots worldwide
- Comprehensive Travel Information 
  - Visa requirements and costs
  - Accommodation options and pricing
  - Safety and security advisories
  - Weather information
  - Top attractions
- Advanced Search & Filtering Find destinations by name, country, or region
- Responsive Design Works perfectly on all devices
- Real-time Data Integrates with multiple APIs for up-to-date information
- Offline Support Fallback data when APIs are unavailable

## APIs Used

- REST Countries API Country information and details
- Open-Meteo Weather API Current weather and forecasts
- ExchangeRate API Currency conversion rates
